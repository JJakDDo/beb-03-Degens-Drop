현재 보유중인 NFT들을 리스트 형식으로 볼 수 있고 거래소에 등록된 NFT들의 목록을 보여주고 등록된 가격만큼 판매자에게 지불하면 NFT들을 구매하는 거래소 웹 어플리케이션입니다. 또한 원하는 이미지 또는 파일 등으로 새로운 NFT를 만들 수 도 있습니다.
